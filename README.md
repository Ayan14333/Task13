# Task13 - (Check if a number is even or odd)

## Project Description
This is a simple beginner-friendly JavaScript program that checks whether a given number is even or odd. It uses basic programming concepts like variables, conditional statements, and operators.

## Features
- Easy to understand logic
- Beginner-friendly code
- Uses if-else condition
- Displays result in console

## How It Works
- The program takes a number
- It uses the modulus operator (%)
- If the remainder is 0 → number is Even
- Otherwise → number is Odd

## Example Output
The number 7 is Odd

## How to Run
- Open any code editor (VS Code recommended)
- Copy the code into a .js file

- Run using:

- node filename.js

- OR open browser console and paste the code
